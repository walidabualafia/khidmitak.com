# ‚ö° Quick Start - Deploy in 5 Minutes

## Option 1: Netlify (Easiest - Recommended)

### Via Netlify CLI

```bash
# Install Netlify CLI
npm install -g netlify-cli

# Navigate to your project
cd path/to/khidmitak

# Login to Netlify
netlify login

# Deploy
netlify deploy

# When prompted:
# - Create & configure a new site: Yes
# - What would you like to deploy: . (current directory)
# - Deploy path: .

# If everything looks good, deploy to production:
netlify deploy --prod
```

Your site is now live at `your-site-name.netlify.app`

### Via Netlify Dashboard (No CLI needed)

1. Go to [app.netlify.com](https://app.netlify.com)
2. Drag and drop your project folder
3. Done! Site is live

## Option 4: Traditional FTP Hosting

### Using FileZilla

1. Download FileZilla: https://filezilla-project.org
2. Open FileZilla
3. Enter host, username, password (from your hosting provider)
4. Connect
5. Navigate to `public_html` or `www` folder on remote
6. Drag all your files from local to remote
7. Wait for upload to complete
8. Visit your domain

### Using Terminal (macOS/Linux)

```bash
# Using SCP
scp -r * username@yourserver.com:/path/to/public_html/

# Using rsync (better for updates)
rsync -avz --delete * username@yourserver.com:/path/to/public_html/
```

## üîß Post-Deployment Setup

### 1. Configure Custom Domain

#### Netlify
```bash
netlify domains:add yourdomain.com
```

Then update DNS records at your registrar:
```
Type: CNAME
Name: www
Value: your-site.netlify.app
```

#### Vercel
```bash
vercel domains add yourdomain.com
```

Follow DNS instructions provided

### 2. Add Environment Variables (if needed)

#### Netlify
```bash
netlify env:set SHOPIFY_TOKEN "your-token"
netlify env:set API_KEY "your-key"
```

#### Vercel
```bash
vercel env add SHOPIFY_TOKEN
vercel env add API_KEY
```

### 3. Set up Shopify (if using e-commerce)

```bash
# No terminal commands needed
# Configure in js/config.js:

const SHOPIFY_CONFIG = {
  storeName: 'your-store',
  storefrontAccessToken: 'YOUR_TOKEN',
  apiVersion: '2024-01',
  domain: 'your-store.myshopify.com'
};
```

Then re-deploy:
```bash
netlify deploy --prod
# or
vercel --prod
```

## üö® Common Issues & Fixes

### Issue: "404 - Page Not Found"

**Fix:** Check that index.html is in the root directory

```bash
# Should look like:
your-project/
  index.html  ‚Üê Must be here
  shop.html
  css/
  js/
```

### Issue: "Images not loading"

**Fix:** Check image paths in HTML
```html
<!-- Wrong -->
<img src="/images/product.jpg">

<!-- Right (for relative paths) -->
<img src="images/product.jpg">

<!-- Or use full URLs -->
<img src="https://yourdomain.com/images/product.jpg">
```

### Issue: "JavaScript not working"

**Fix:** Check browser console for errors
- Right-click ‚Üí Inspect ‚Üí Console
- Fix any errors shown
- Common: File paths incorrect

### Issue: "CSS not loading"

**Fix:** Verify CSS path
```html
<!-- Check this line in HTML -->
<link rel="stylesheet" href="css/styles.css" />
```

### Issue: "Cart not saving"

**Fix:** Check localStorage is enabled
- Works locally but not on some browsers in incognito mode
- Test in normal browsing mode
